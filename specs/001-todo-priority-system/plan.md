## Summary

Add a three-level priority system (High / Medium / Low) to the existing vanilla JS todo app. Each todo gains a `priority` field (default: `medium`). The UI adds a priority selector to the creation form, a priority badge on each todo item, an in-place priority editor on existing todos, and a "Sort by Priority" toggle that follows the existing due-date sort pattern.
